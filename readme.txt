== Description ==

BlogShotR is a very easy useable plugin, wich helps you after activating to show the commentators Blog or Homepage with a very mini Screenshot right in the comments.
It is like the easygravatars plugin and very easy to use. You can set with and hight of the Shot at admin interface

= Localization =

* **English** \* Author: [Ahmet Topal](http://basicblogger.de/2008/11/16/wp-plugin-blogshotr/)

== Installation ==

1. Download and extract the Plugin
2. Just upload the .php file and activate it
3. More under setting/BlogShotR
